Mohammed Islam

My facination about how computer works was rooted into my mind since my childhood. Always my curiousity about this magic box was really really high. I always had dream to be a coder or computer programmer. But for some reason that dream never was materialized for such a long time. Finally after a couple of years—dabbling in web tutorials here and spending time on FreeCodeCamp.org and Portnov Computer School there—I decided to commit to Northwestern University's coding boot camp.

I'm leveraging my know-how of syntax and language and to master the rules of HTML, CSS, JavaScript, and its many libraries. I want to combine my new skills with my enormous interest to coding and move into full-stack web production.

My goal is to meet and work with a team of like-minded people where I can ideate freely and use my expertise to achieve excellence or build something truly remarkable and beautiful.

Portfolio

Fresh Farm Recipes

Tools used: jQuery, Bootstrap, Edamam API, USDA farmer's market API, Leaflet.js

I worked with a small team to build an app that provides recipe suggestions based on raw ingredients and helps users find nearby farmer's markets to procure those ingredients. My primary work was on the look and functionality of the site's Leaflet.js map and the results brought back from the USDA's farmer's market API.

Train-Scheduler

Tools used: jQuery, Firebase

Using Firebase for back-end server processes, I built this Train-Scheduler.

GifTastic GIF Finder

Tools used: jQuery, localStorage, Giphy API

This is a GIF search engine for movies, with pause-and-play functionality for each GIF and the option for each user to save favorited GIFs from session to session. I put it together using the Giphy API and localStorage.

Crystal Collector

Tools used: jQuery

This is a simple Crystal Collector game, built with jQuery.

Timed Trivia Game

Tools used: jQuery, setInterval, setTimeout

This is a fun little general knowledge trivia game I built to practice working with setInterval and setTimeout more. Users get 20 seconds to answer all of the questions, and their score is tallied and presented to them.
